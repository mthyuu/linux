## chmod - "change mode/perms"
┌──────────────────────────────────────────────────────┐
│            The r, w, and x bits                      │  
├─────────┬───────┬────────────────────────────────────┤  
│ Letter  │ Value │ Meaning                            │  
├─────────┼───────┼────────────────────────────────────┤  
│ r       │ 4     │ read  (can view contents)          │  
│ w       │ 2     │ write (can modify/delete)          │  
│ x       │ 1     │ execute (run or enter dir)         │  
│ -       │ 0     │ no permission                      │  
└─────────┴───────┴────────────────────────────────────┘
┌──────────────────────────────────────────────────────┐
│           Translating to numbers (chmod ###)         │
├──────────┬────────┬────────┬────────┬────────────────┤
│ Who      │ Symbol │ Binary │ Dec    │ Meaning        │
├──────────┼────────┼────────┼────────┼────────────────┤
│ Owner    │ rwx    │ 111    │ 7      │ Full access    │
│ Group    │ r-x    │ 101    │ 5      │ Read + execute │
│ Others   │ r--    │ 100    │ 4      │ Read only      │
└──────────┴────────┴────────┴────────┴────────────────┘
